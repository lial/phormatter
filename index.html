Phone formatter

This Node.JS module was ported from php library (https://github.com/mrXCray/PhoneCodes) It formats phone number in pretty, more readable string, depends on country code and length of phone number.

Installation

$ npm i phormatter

Using

Module has only one function format, that can be call like this:

var phormatter = require('phormatter');

var phone = '74991234567';
phone = phormatter.format(phone);
console.log(phone); //Output 7 (499) 123-45-67

phone = phormatter.format(phone, ' ', true);
console.log(phone); //Output +7 (499) 123 45 67

phormatter.format(phone, divider, plusForce, convertCharNumber);
